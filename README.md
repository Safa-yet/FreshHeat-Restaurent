# 🍽️ FreshHeat – Restaurant Website

A modern and responsive restaurant landing page built using Tailwind CSS. This project showcases a clean and attractive UI design for food and restaurant businesses.

---

## 📖 Project Overview

FreshHeat is a stylish restaurant website designed to present food items, menu sections, and restaurant services in a visually appealing way. The goal of this project is to create a smooth and engaging user experience for customers.

## ⚙️ Installation & Setup Guide

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/Safa-yet/FreshHeat-Restaurent.git

# Go to project folder
cd FreshHeat-Restaurent

# Install dependencies
npm install

# Run the project
npm run dev
```
